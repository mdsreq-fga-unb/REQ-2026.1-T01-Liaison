# Bem-vindo ao Liaison

O **Liaison** é uma plataforma de conexão entre estudantes universitários e organizações do terceiro setor, projetada para facilitar a descoberta, inscrição e certificação de atividades de voluntariado de forma centralizada, confiável e academicamente reconhecida.

Este projeto está sendo desenvolvido pelo **Grupo Dona Izeti** na disciplina de **Engenharia de Requisitos de Software** (UnB - 2026/1).
